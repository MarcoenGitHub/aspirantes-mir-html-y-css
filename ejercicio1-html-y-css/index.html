Ensalada con pollo

Esta es una opcion nutritiva y balanceada

Ingredientes

...Los ingredientes son:

  • Pechuga de pollo
  • lechuga romana
  • Lechuga morada
  • Lechuga crespa
  • Cebolla morada
  • Zanahoria
  • Pimenton rojo
  • Maiz
  • Alverja verde o chicharo

Preparacion

...La prepararemos asi:

  1. agregar una capa de lechugas
  2. agregar una capa de pollo
  3. agregar otra capa de lechugas
  4. agregar una capa Zanahoria y Pimenton
  5. agregar el maiz y aceite de olivas
